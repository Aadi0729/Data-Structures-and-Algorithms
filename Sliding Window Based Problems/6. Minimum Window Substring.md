Problem

Given two strings s and t, return the smallest substring of s that contains all characters of t (including their frequencies).
If no such substring exists, return an empty string.

Key Idea

Use Sliding Window.

We expand the window until it contains all required characters, then shrink it to find the minimum valid window.

Window Validity

Maintain two maps:

needMap → frequency of characters required from t

window → frequency of characters inside the current window

Track:

need = number of unique characters in t
have = number of characters currently satisfying their required frequency

Window becomes valid when

have == need
Algorithm

Build frequency map for t.

Use two pointers left and right.

Expand the window using right.

Update frequency counts.

If a character meets its required frequency → increment have.

When have == need:

Update minimum window length.

Shrink from left.

If shrinking breaks the requirement → decrement have.

Continue until the entire string is processed.

Sliding Window Pattern
Expand window → until valid
Shrink window → while valid
Update minimum answer
Time Complexity
O(n)

Each character enters and leaves the window at most once.

Space Complexity
O(m)

Where m is the number of unique characters in t.

```
class Solution {
public:
    string minWindow(string s, string t) {
        int n = s.size();

        unordered_map<char, int> requiredFreqMap;
        unordered_map<char, int> freqMap;

        for(auto ch : t)
            requiredFreqMap[ch]++;
        
        int requiredChar = requiredFreqMap.size();
        
        int left = 0;
        int right = 0;

        int minLen = INT_MAX;

        int count = 0;
        int start = 0;

        while(right < n) {
            freqMap[s[right]]++;

            if(requiredFreqMap.count(s[right]) && requiredFreqMap[s[right]] == freqMap[s[right]])
                count++;

            while(requiredChar == count) {
                if(right - left + 1 < minLen) {
                    minLen = right - left + 1;
                    start = left;
                }

                freqMap[s[left]]--;

                if(requiredFreqMap.count(s[left]) && freqMap[s[left]] < requiredFreqMap[s[left]])
                    count--;
                
                left++;
            }

            right++;
        }

        if(minLen == INT_MAX)
            return "";
        
        return s.substr(start, minLen);
    }
};
```
