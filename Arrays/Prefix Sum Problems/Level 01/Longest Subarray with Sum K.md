## Problem Statement

You are given an integer array nums of size n and an integer k.

A subarray is a contiguous part of the array.

Your task is to find the length of the longest subarray whose sum is exactly equal to k.

If no such subarray exists, return 0.

**Example 1**
*Input:*
nums = [10, 5, 2, 7, 1, 9]
k = 15

*Output:*
4

*Explanation:*
The longest subarray with sum 15 is:

[5, 2, 7, 1]

Length = 4

**Example 2**
*Input:*
nums = [1, -1, 5, -2, 3]
k = 3

*Output:*
4

*Explanation:*
The longest subarray with sum 3 is:

[1, -1, 5, -2]

Length = 4

**Example 3**
*Input:*
nums = [1, 2, 3]
k = 7

*Output:*
0

*Explanation:*
No subarray has sum 7.
